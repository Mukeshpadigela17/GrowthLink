Project 1

Description of project 1.
The Personal Fitness Tracker is a Python-based application designed to help individuals monitor and manage their fitness activities, diet, and overall health in an accessible and user-friendly manner. Unlike expensive wearable-based fitness solutions, this tracker provides a cost-effective and customizable alternative, enabling users to track their progress without requiring additional hardware or subscriptions. The system allows users to log workouts, track calorie intake, monitor hydration levels, and analyze progress over time through graphical visualizations. The project leverages Python libraries such as Tkinter/PyQt for GUI, SQLite for data storage, and Matplotlib/Seaborn for data visualization.

Project 2

Description of project 2.
In many countries, traffic violations—such as riding without a helmet—pose a significant risk to road safety. To address this issue, I developed a real-time Helmet Detection and Number Plate Extraction System using AI and Computer Vision. This project automates the process of detecting helmet-less riders and identifying their vehicles by extracting number plates, ultimately assisting traffic authorities in enforcing laws more efficiently. The system captures live video footage or processes images from surveillance cameras. Using YOLO (You Only Look Once), an advanced deep-learning object detection model, it identifies motorcyclists without helmets. Simultaneously, OCR (Optical Character Recognition) and OpenCV extract the number plates of violators, enabling automatic challan (fine) generation.
